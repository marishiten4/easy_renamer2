# Easy Renamer

## 概要
Stable Diffusionで生成した画像を効率的にリネームするためのWebツール。

## 機能
- 画像フォルダの選択
- メタデータ抽出
- カスタマイズ可能なリネーム
- バックアップ機能

## インストール

### 依存関係のインストール
```bash
pip install -r requirements.txt
```

### 起動方法
```bash
streamlit run src/rename_tool.py
```

## 開発環境
- Python 3.8+
- Streamlit
- Pillow
